
## Про роботу

Програма — це онлайн-каталог музеїв. Користувач може зайти на сайт, щоб побачити список усіх музеїв,
відсортувати їх за країною або типом (наприклад, тільки "Художні" або "Історичні") та переглянути детальну інформацію про кожен заклад.
Також програма дозволяє додавати нові музеї до загального списку через зручну форму.

## Використання Ші

У роботі використовувався штучний інтелект Gemini.Він був використаний для написання коду.

## Робота програми

Технічно система працює так: веб-сервер на базі Flask обробляє запити користувача, а база даних PostgreSQL зберігає всю інформацію.
Головна особливість — автоматизація: як тільки ви додаєте або видаляєте музей, спеціальний механізм у базі даних (тригер)
миттєво перераховує загальну кількість музеїв для відповідної країни, тому дані завжди актуальні.

## Що треба для запуску програми

Підготуйте середовище: Встановіть мову програмування Python та систему бази даних PostgreSQL.Для їх використання встановіть PyCharm та pgAdmin4.

    Інструкції для встановлення PostgreSQL:
    https://neon.com/postgresql/getting-started
    Інструкції для встановлення Python:
    https://www.jetbrains.com/pycharm/download/?section=windows

Налаштуйте базу: У pgAdmin4 створіть порожню базу даних з назвою Web_app та вкажіть пароль від неї у файлі app.py (у рядку SQLALCHEMY_DATABASE_URI).

Встановіть залежності: Відкрийте термінал у папці проєкту і виконайте команду pip install flask flask-sqlalchemy psycopg2, щоб програма отримала необхідні компоненти.
Або під час запуску програми вам висвітиться чого не вистачає і PyCharm сам все докачає.

Запустіть: Введіть команду python app.py або зелену стрілочку . Програма сама створить таблиці та заповнить їх прикладами музеїв,
після чого ви зможете відкрити сайт у браузері за посиланням, яке з'явиться в терміналі (зазвичай це http://127.0.0.1:5000/).
